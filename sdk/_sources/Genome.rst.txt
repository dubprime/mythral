Genome
======

Genome is the a class that encapsulates a representation of a GAIuS agent topology.
This class keeps track of primitive nodes, manipulative nodes, and provides a mechanism for visualizing a topology.
Automatically created by AgentClient when connecting to a GAIuS agent.
